Below is a table summarizing some quirks in μL in the book of Job. For more details, see the document, “Quirks in μL in Job.”

Each entry below takes the following form:

  • # (hash sign) (number sign) is a link to more details on this quirk

  • c:v is the chapter and verse of the book of Job
